# 2.使用说明
所有方块信息都会实时记录，查询也十分方便，只需要一下几步  
1.在你想要查询的位置放一个基岩（badrock）  
2.然后破坏这个基岩  
3.信息已经显示在对话框中了  
显示的格式为:名字 时间 方块名 类型
